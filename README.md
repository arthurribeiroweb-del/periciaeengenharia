# AB ENGENHARIA — Landing Page (React + Vite)

## Como rodar
1. Instale Node.js 18+.
2. No terminal:
   ```bash
   npm install
   npm run dev
   ```
3. Abra o endereço que o Vite mostrar (ex.: http://localhost:5173).

## Como gerar build
```bash
npm run build
npm run preview
```

## Observações
- Tailwind é carregado via CDN no `index.html` (sem configuração).
- Dependências: framer-motion, lucide-react, jspdf, jspdf-autotable.
- Edite dados em `src/App.jsx` (telefone, e-mail, CREA, serviços, etc.).
